Hamburgueria : Planilha de Requisitos e Regras de Negócio

Para este projeto, desenvolvi a documentação completa de uma hamburgueria, organizando todos os elementos essenciais para o seu planejamento e operação. Criei um documento detalhado contendo:

Identidade visual: Inseri o logo e o nome da hamburgueria, destacando sua proposta de valor (como conceito e público-alvo).

Requisitos funcionais: Liste as funcionalidades necessárias para o negócio, como sistema de pedidos, cadastro de clientes e controle de estoque.

Requisitos não-funcionais: Defini critérios de performance, segurança e usabilidade para garantir uma experiência eficiente.

Regras de negócio: Estabeleci normas operacionais, como políticas de promoções, horários de funcionamento e condições de personalização dos lanches.

Utilizei uma planilha para estruturar as informações de forma clara e acessível, facilitando a visualização e o alinhamento com a equipe. Essa atividade reforçou minha capacidade de organizar requisitos e regras de negócio, essenciais para qualquer projeto bem-sucedido.

Habilidades : H4, H7, H8

3º Trimestre

Desenvolvimento, Teste e Manutenção de Sistemas

A atividade consistiu no desenvolvimento individual de um Sistema de Cadastro de Livros para a Biblioteca Escolar. Todo o projeto foi realizado utilizando Django e Python, desde a construção do backend até a criação das páginas do sistema. A documentação inicial apresentou o nome do projeto, sua descrição geral, funcionalidades principais e as tecnologias empregadas. Em seguida, foi implementado um sistema completo capaz de listar livros, cadastrar novos títulos e excluir registros, utilizando um banco de dados integrado ao Django. Depois da versão funcional pronta, foram inseridos erros propositais para a etapa de testes, simulando falhas de lógica, interface e consultas ao banco. O projeto foi publicado no GitHub com duas branches: uma contendo o sistema funcionando corretamente e outra com os erros intencionais. A atividade proporcionou prática real de desenvolvimento, organização técnica, testes e manutenção de software, além do uso profissional de um framework backend.

Habilidades : C1 - H5, H6, H7, H10
